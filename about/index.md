# Cheng Xiaofan

> About Cheng Xiaofan, a student at the Computer Network Laboratory, The University of Tokyo.


I use this website as a place to organize my learning process, paper reading notes, technical reflections, and research ideas.

My current interest lies in datacenter systems, especially the intersection of computer networks and storage systems. I am interested in how data movement, storage access patterns, network bottlenecks, and system-level resource management interact in large-scale cloud environments.

At the moment, I mainly read papers on datacenter networks, distributed storage systems, cloud infrastructure, operating systems, and network-storage co-optimization. Some notes are polished, while others are simply traces of my learning process. I hope to use this space to think more clearly, write more consistently, and gradually build a deeper understanding of systems research.
